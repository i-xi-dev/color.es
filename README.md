# @i-xi-dev/color

A JavaScript immutable object that represents the color.


## Requirement

| Chrome | Edge | Firefox | Safari | Deno | Node.js |
| :---: | :---: | :---: | :---: | :---: | :---: |
| ✅ | ✅ | ✅ | ✅ | ✅ | ✅ |


## Installation

### npm

```console
$ npm i @i-xi-dev/color@5.2.3
```

```javascript
import { RgbColor } from "@i-xi-dev/color";
```

### CDN

Example for Skypack
```javascript
import { RgbColor } from "https://cdn.skypack.dev/@i-xi-dev/color@5.2.3";
```
